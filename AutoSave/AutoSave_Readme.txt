---- Unity AutoSaving ----

1. Import the Unity_AutoSavePackage package into Unity or drag and drop the scripts into an Editor folder.
2. Upon import a Scriptable Object called AutoSaveConfig should be created in the same folders as the scripts.
3. By default, AutoSaveConfig has been set to save every 10 minutes. This can be adjusted by selecting the AutoSaveConfig and adjusting the number.
4. Unity should now start autosaving every 10 minutes.
5. If you do not see any console messages about autosaving please enter playmode or restart unity and the tool should initialize or go through Tools/AutoSaving/Reset AutoSaving
